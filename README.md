# 🚀 Focus Lock App

A simple productivity app to block distracting apps and improve focus.

---

## 🎯 Goal

Build a real-world startup-like project where team members collaborate and learn together.

---

## 👥 Team Roles

* 💻 Coders: Build app (frontend + backend)
* 🎨 Designers: Create UI/UX
* ✍️ Content: App description, features, branding
* 📢 Managers: Manage tasks & progress

---

## 🔥 Features

* App blocker system
* Study timer
* Daily streak tracking
* Motivation quotes

---

## 🛠 Tech Stack

* Frontend: React / Flutter
* Backend: Node.js / Firebase
* Design: Figma
