# Contributing to Focus Lock App

Thanks for your interest in contributing! 🙌

## How to Contribute

- **Developers** — Pick any open task from `docs/tasks.md` and start building
- **Designers** — Add your UI/UX work inside the `design/` folder
- **Content Creators** — Help with app name, tagline, social media content
- **Marketing** — Share ideas on how we can reach users

## Getting Started

1. Fork this repository
2. Create a new branch for your work
3. Make your changes
4. Open a Pull Request and describe what you did

## Questions?

Just open an Issue or reach out directly. No question is too small!
